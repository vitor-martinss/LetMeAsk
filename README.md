## 💻 Getting started


### Requirements


**Clone the project and access the folder**

```bash
$ git clone https://github.com/vitor-martinss/LetMeAsk.git
```

**Follow the steps below for Web Version**

```js
  # Go to src folder
  cd LetMeAsk.static 

  # Install the dependencies
  $ yarn

  # Start project
  yarn start
```
